# Admin Community Resources

Community-shared Claude Code skills and Gainsight admin utilities. Everything here is shared out of kindness — take what's useful, adapt it to your setup.

## What's here

### [`skills/`](skills/)
Claude Code skills you can drop into your own setup. Each skill lives in its own subfolder with a `SKILL.md` (the skill definition Claude reads) and any supporting files. See the [skills index](skills/README.md) for a quick overview.

### [`scripts/`](scripts/)
Browser console scripts for Gainsight admins — run directly in your browser's DevTools console against your Gainsight tenant.

## Contributing

- Remove any PII or org-specific configuration before uploading (tenant URLs, API keys, personal names, account names)
- All materials assumed under MIT license
- Skill files should follow the `skills/<skill-name>/SKILL.md` structure
- Scripts should go in `scripts/` with a short description at the top of the file
