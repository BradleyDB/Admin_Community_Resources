# Skills

Each skill lives in its own subfolder. Drop the `SKILL.md` into your `.claude/skills/<skill-name>/` directory to install it.

| Skill | Folder | Trigger phrases | What it does |
|---|---|---|---|
| lfg | [lfg/](lfg/) | "lfg", "let's go", "start session", "boot up" | Session init — loads memory, surfaces active work, Jira, Slack, Gmail, and Calendar |
| wrap-up | [wrap-up/](wrap-up/) | "weekly wrap up", "wrap up the week" | Personal weekly status across Calendar, Slack, meeting transcripts, and Jira |
| grill-me | [grill-me/](grill-me/) | "grill me", "interview me" | Relentless interview that extracts and checkpoints plans/designs into a durable capture file |
| cs-sentiment | [cs-sentiment/](cs-sentiment/) | "run my sentiments", "sentiment update", "which accounts need sentiment" | CS Sentiment drafting agent for Gainsight CSMs — researches accounts and drafts sentiments for approval |
| cs-exec-briefing | [CS-Briefing/](CS-Briefing/) | "run briefing", "generate briefing", "cs briefing" | Weekly CS executive briefing from Staircase AI signals, delivered via Slack Canvas |
